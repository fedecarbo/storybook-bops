# Check Consultees Consulted (Assessment) — User Journey

During the assessment stage, the case officer confirms that all statutory consultees have been consulted. The consultee selection and emailing happened in Stage 2 (Consultation) — this task is a review and confirmation step. One actor: **case officer**.

**Who:** Case officer
**Where:** Assessment task list → Check application → Check consultees consulted

The case officer opens the task from the assessment task list. The page shows a table of all consultees grouped by constraint, with their current consultation status. Most consultees will have been emailed during Stage 2 — their statuses show whether they are awaiting a response or have already responded.
