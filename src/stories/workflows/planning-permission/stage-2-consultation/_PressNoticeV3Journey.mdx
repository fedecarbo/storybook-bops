# Press Notice v3 — Multi-notice Support

v3 builds on v2's overview-first approach and adds support for **multiple press notices per application** using GDS Summary Cards. Same two layout variants as site notice v3.

**What's new in v3:**
- Overview page displays notices as **GDS Summary Cards**
- **"Add another press notice"** button
- **Variant A**: Full detail on each card (reasons, timeline, evidence inline)
- **Variant B**: Compact cards with "View details" link to a detail page
