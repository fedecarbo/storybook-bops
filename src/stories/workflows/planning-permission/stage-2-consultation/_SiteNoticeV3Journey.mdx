# Site Notice v3 — Multi-notice Support

v3 builds on v2's overview-first approach and adds support for **multiple site notices per application** using GDS Summary Cards. Two layout variants are provided for comparison.

**What's new in v3:**
- Overview page displays notices as **GDS Summary Cards** (works for one or many)
- **"Add another site notice"** button for creating additional notices
- **Variant A**: Full detail on each card (timeline, evidence, actions inline)
- **Variant B**: Compact cards with a "View details" link to a separate detail page
