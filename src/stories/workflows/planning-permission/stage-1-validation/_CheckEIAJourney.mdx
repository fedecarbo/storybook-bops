# Check EIA — User Journey

This documents the journey for checking whether an Environmental Impact Assessment (EIA) is required under the Town and Country Planning (Environmental Impact Assessment) Regulations 2017. This is a **single-actor** task — only the **case officer** is involved. If an EIA is required, the determination period is extended from 8 weeks to 16 weeks.

A guidance link to GOV.UK's EIA screening checklist opens in a new tab to help the officer make their determination.

Below this, a Yes/No radio question asks whether an Environmental Impact Assessment is required.

**Where:** Check EIA form

The officer reviews the application documents and the GOV.UK guidance, then selects their answer.

### If EIA is not required

The officer selects **No** — no additional fields are needed. They click "Save and mark as complete".

<Canvas of={CheckEIAStories.SelectedNo} />

### If EIA is required

The officer selects **Yes**. A hint explains that the determination period will be extended to 16 weeks. Three optional fields appear for public access to the Environmental Statement:

- **Email address** — where the public can request a copy
- **Address** — where the public can view or request a copy (must include fee if provided)
- **Fee** — cost to obtain a hard copy (must include address if provided)

The officer can fill in all fields, some, or none — all three are optional. However, if either the address or fee is provided, both must be filled in.
