# Confirm CIL — User Journey

This documents the journey for confirming Community Infrastructure Levy (CIL) liability. The case officer reviews data from the PlanX submission (proposed floor area and a liability recommendation) and confirms whether the application is liable for CIL. This is a **single-actor** task — only the **case officer** is involved.

**Who:** Case officer
**Where:** Validation task list → Confirm Community Infrastructure Levy (CIL)

The case officer opens the task from the validation task list. They see information from the PlanX submission: the proposed net floor area in square metres and a recommendation of whether the application is likely liable or exempt from CIL.

Below this, a Yes/No radio question asks the officer to confirm CIL liability. If PlanX recommends "liable", the Yes option includes a **(recommended based on submission data)** label.

<GdsCanvas of={ConfirmCILStories.InitialView} />

If PlanX recommends the application is exempt, the "No" option gets the recommended label instead.

<GdsCanvas of={ConfirmCILStories.RecommendedExempt} />

If no PlanX data is available (e.g. the application was submitted outside PlanX), a fallback message is shown and the officer must make the determination without a recommendation.
